一.github

  • 1.1:github基本介绍
    https://github.com/
    Github是一个git服务的提供商(提出社交化编程)
  • 1.2：创建用户
    在github中创建一个账户，并且创建一个空的远程仓库。

  • 1.3：远程提交
    Git remote -v
    git remote add origin https://github.com/kuaixiangai/git-demo.git
    git remote rm origin（删除远程仓库信息）
    git push -u origin master

  • 1.4：远程拉取
    Git clone （克隆相关的代码）
    Git pull origin master （拉取最新的代码）

2.4 hexo工具

  • 1：使用npm安装hexo
    npm install -g hexo
  • 2：初始化hexo
    Hexo init blog

  • 3：生成静态页面
    hexo generate

  • 4： 启动本地服务，进行文章预览调试
    hexo server

  • 5：修改_config.yml文件

    deploy:

    type: git
repository:*****
branch: gh-pages

  • 6:下载安装 hexo-deployer-git
    npm install hexo-deployer-git –save

  • 7：发布到github
    hexo deploy –generate

注意：

打开网站会出现乱掉的情况，由于需要配置相应的路径

  • 修改_config.yml文件
    root: /blog/
  • 重新发布
    hexo clean
    hexo deploy –generate
